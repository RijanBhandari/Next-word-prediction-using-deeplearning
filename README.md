# Next Word Prediction with Deep Learning

A Natural Language Processing (NLP) project that uses deep learning to predict the most likely subsequent word in a given sentence.

## 🛠️ Project Features
* **Deep Learning Model:** Built using `TensorFlow` and `Keras` with an **LSTM (Long Short-Term Memory)** architecture to understand sequential context.
* **Text Processing:** Implements tokenization, sequence padding, and n-gram generation to prepare raw text for training.
* **Embeddings:** Uses an Embedding layer to represent words in a dense vector space, capturing semantic relationships.
* **Interactive Prediction:** Features a script to provide a "seed text" and receive an automatically generated next-word prediction.

## 🚀 How to Use
1. Open the `Next_Word_Prediction_with_Deep_Learning_in_NLP (1).ipynb` notebook.
2. Provide a source text or dataset for the model to learn from.
3. Run the training cells to fit the LSTM model.
4. Input a partial sentence (e.g., "Ram has a") to see the model predict the next word.
